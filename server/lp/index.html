AWT is a Service (SaaS) for Web site Testing, Automation and Monitoring by user-defined Web Browser actions.

Create Test. Test comprises of Test Actions. The Test Action may be as "open site", "click the link", "enter the value into the input", etc.

Run the Test. The Test will be executed server-side on a chosen Web Browser. For visual Test Actions Screenshots will be taken.

Set up a Schedule and E-Mail reports. The Test will be periodically executed on a Schedule, results will be sent by E-Mail.

Use Cases

Web site development. Auto-testing. Check the elements are present on the page and nothing is lost.

Cross-browser validation. Mobile & Desktop.

Website monitoring. Cross-site integration link monitoring. Periodic tests with notifications.

Data mining. Check site ranking. Check the AD is still on the board. Check market prices.

Features

  • Auto-Testing + Scheduling + Notifications = Monitoring
  • Variables (variable substitution in Test Action parameters)
  • Screenshots

Supported Test Actions

  • Open URL
  • Check element exists
  • Enter value into input
  • Click
  • Set value to arbitrary element attribute or text (Modify XPATH)
  • Check browser URL (Match URL)
  • Check browser Title (Match Title)
  • Save XPATH/URL/Title into variable, apply RegExp to variable
  • Request any other Test Action and we implement it in 2 days

Supported Web Browsers

  • Firefox
  • Google Chrome / Chromium / Mobile Emulation
  • Internet Explorer 6/7/8/9 (beta)
  • PhantomJS
  • Request a particular browser version and we set it up

Supported Notifications

  • E-Mail
  • XMPP/Jabber (beta)
  • SMS/Text (beta)
