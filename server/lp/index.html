Create a Test. The Test comprises of Test Actions. Test Action may be like "open site", "click the link", "enter the value into an input", etc. See supported Test Actions in Features.

Run the Test. The test will be executed automatically on a chosen Web Browser. See supported Web Browsers in Features.

Set up a Schedule and E-Mail reports. The Test will be executed periodically on a Schedule, results will be sent by E-Mail.

Use Cases

Web site development. Auto-Testing.

Web site monitoring. Complex third-party integration link monitoring.

Data mining. Check site ranking. Check the advertisement is still on the board. Check market prices.

Features

  • Auto-Testing + Scheduling + Notifications = Monitoring
  • Variables (variable substitution in Test Action parameters)

Supported Test Actions

  • Open URL
  • Check element exists
  • Enter value into input
  • Click
  • Set value to arbitrary element attribute or text (Modify XPATH)
  • Check browser URL (Match URL)
  • Check browser Title (Match Title)
  • Save XPATH/URL/Title into variable, apply RegExp to variable
  • Request any other Test Action and we implement it in 2 days

Supported Web Browsers

  • Firefox
  • Google Chrome / Chromium
  • Internet Explorer 6/7/8/9 (beta)
  • Request a particular browser version and we set it up

Supported Notifications

  • E-Mail
  • XMPP/Jabber (beta)
  • SMS/Text (beta)
