Установка
=========

Зависимости
-----------
* php5-json
* php5-mysql
* php5-xsl
* php5-gd
* php5-curl (для webmoney)
* php-pear
* php5-cli (для cron)

* xvfb
* dwm
* scrot
* jre (openjdk)
* x11vnc

* firefox
* chromium-brwser
* flashplugin-installer

Регламент
---------
1. task_types.sql
2. Зарегистрировать пустой логин и ввести сложный пароль.
3. Добавить demo_subscription(.sql) для демо-логина.
4. Выставить undeletable в settings для демо-логина.
